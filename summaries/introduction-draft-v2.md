## Ember: Low Tail Latency Cloud-Native Vector Search

- In the LLM era, cloud vector databases have become a foundational infrastructure, and also face new challenges, such as continuously-growing workload scale.
- First-generation cloud vector databases are traditional provisioned solutions hosted on cloud VMs without architectural change.
    - They were designed for small-scale workloads.
    - Always-on compute resources + local SSDs.
    - These solutions can achieve good performance, but they are becoming more and more incompetent for LLM applications.
